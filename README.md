# STM32F4 Digital Band-Pass Filter (ADC → IIR Filter → DAC)

## Описание

Проект реализует цифровой полосовой фильтр (Band-Pass Filter) на микроконтроллере STM32F4 с использованием библиотеки libopencm3.

Сигнал непрерывно считывается через АЦП (ADC1), обрабатывается цифровым IIR-фильтром 4-го порядка и выводится через ЦАП (DAC Channel 2). Обработка выполняется по таймерному прерыванию с частотой дискретизации 17.5 кГц.

### Схема обработки сигнала

```text
Analog Input
      │
      ▼
   ADC1 (PA0)
      │
      ▼
 IIR Band-Pass Filter
      │
      ▼
   DAC1 CH2 (PA5)
      │
      ▼
 Analog Output
```

| Параметр              | Значение             |
| --------------------- | -------------------- |
| MCU                   | STM32F4              |

| Вход                  | ADC1 Channel 0 (PA0) |
| Выход                 | DAC Channel 2 (PA5)  |

| Порядок фильтра       | 4                    |

## Структура программы

### Коэффициенты фильтра

Фильтр реализован в прямой форме через разностное уравнение.

Числитель (Feedforward):

```c
b0 =  0.013202172
b1 =  0.0
b2 = -0.026404344
b3 =  0.0
b4 =  0.013202172
```

Знаменатель (Feedback):

```c
a1 = -3.6585689
a2 =  5.0865269
a3 = -3.1905211
a4 =  0.7630810
```

Для работы фильтра хранятся предыдущие входные и выходные отсчёты:

```c
x1 ... x4
y_1 ... y_4
```

---

### Функция `filter()`

```c
static float filter(float x)
```

Выполняет вычисление одного отсчёта полосового фильтра.

Алгоритм:

1. Вычисляет новое значение выхода по разностному уравнению.
2. Сдвигает историю входных отсчётов.
3. Сдвигает историю выходных отсчётов.
4. Возвращает новое значение.

---

### Функция `read_adc()`

```c
static uint16_t read_adc(uint8_t ch)
```

Выполняет одиночное преобразование выбранного канала ADC.

Последовательность работы:

1. Настройка канала преобразования.
2. Запуск ADC.
3. Ожидание завершения преобразования.
4. Возврат результата (0–4095).

---

## Таймерное прерывание

Обработка сигнала выполняется в обработчике:

```c
void tim2_isr(void)
```

Каждое прерывание выполняет:

1. Считывание значения ADC.
2. Нормализацию диапазона:

```text
0..4095 → -1..1
```

3. Пропускание через IIR-фильтр.
4. Обратное преобразование:

```text
-1..1 → 0..1
```

5. Ограничение результата (clipping).
6. Преобразование в 12-битное значение DAC.
7. Вывод через ЦАП.
8. Переключение состояния светодиода.

## Настройка периферии

### Clock

```c
clock_setup()
```

Настраивает:

* PLL на 168 MHz
* Тактирование GPIO
* ADC1
* DAC
* TIM2

---

### GPIO

```c
gpio_setup()
```

Настраивает:

* PD12 как выход для светодиода.

---

### ADC

```c
adc_setup()
```

Настраивает:

* PA0 как аналоговый вход.
* ADC1 в одиночном режиме.
* Время выборки 84 такта для снижения шумов.

---

### DAC

```c
dac_setup()
```

Настраивает:

* PA5 как аналоговый выход.
* DAC Channel 2.

---

### TIM2

```c
tim_setup()
```

Используется как источник частоты дискретизации.

Параметры:

```text
Timer clock = 1 MHz
Fs = 17.5 kHz
```

Настройки:

```c
Prescaler = APB1 / 1 MHz
Period = 1000000 / 17500
```

Каждое переполнение таймера вызывает прерывание и обработку одного отсчёта сигнала.

---

## Главный цикл

```c
int main(void)
```

Последовательность запуска:

```c
clock_setup();
gpio_setup();
adc_setup();
dac_setup();
tim_setup();
```

После инициализации программа полностью работает через прерывания:

```c
while (1) {
}
```

Основная нагрузка выполняется в обработчике TIM2.

---

## Использование

### Входной сигнал

Подключить аналоговый сигнал к:

```text
PA0 (ADC1_IN0)
```

### Выходной сигнал

Снимать обработанный сигнал с:

```text
PA5 (DAC_OUT2)
```

### Индикация работы

Светодиод:

```text
PD12
```

переключается при каждом срабатывании таймера и позволяет контролировать выполнение цикла обработки.

---

## Принцип работы

```text
ADC Sample
     │
     ▼
Normalization
     │
     ▼
Band-Pass IIR Filter
     │
     ▼
Clipping
     │
     ▼
DAC Output
```

Таким образом реализуется цифровая обработка сигнала в реальном времени с фиксированной частотой дискретизации 17.5 кГц.
